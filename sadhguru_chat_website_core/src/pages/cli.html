CLI & Ollama

Install Python dependencies (see requirements.txt in the GitHub repository), set Ollama env vars, then: 

# One-shot question
python sadhguru_chat.py --question "What is inner engineering?"

# Interactive session
python sadhguru_chat.py --interactive

# Optional styles (CLI only)
python sadhguru_chat.py --interactive --opc
python sadhguru_chat.py --interactive --sans

# Choose a model id known to your Ollama server
python sadhguru_chat.py --question "Hello" --model llama3

--opc enables “Yogic Code” (playful compressed phrasing). --sans asks for a Sanskrit + Tamil blend with English gloss. These modes extend the base prompt defined in sadhguru_chat.py.
