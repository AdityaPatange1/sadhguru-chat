What this project is

Sadhguru Flow is a small Python program that sends your questions to a large language model with a fixed system prompt so replies read like Sadhguru Jaggi Vasudev: poetic, grounded, and inward-pointing.

The CLI uses Ollama (with OLLAMA_HOST and OLLAMA_API_KEY from your environment) so you can pick any model your server exposes — for example gpt-oss:20b as a default.

This website is a single-page app: navigation swaps content without reloading the page. The chat card on Home uses WebLLM so inference happens in your tab via WebGPU, not on our servers.

Browse or clone the full project on GitHub: github.com/AdityaPatange1/sadhguru-flow
